# 🐍 Python Quest

An interactive Python learning app for anyone who already knows JavaScript or Unity C#. No installs needed — just open `index.html` in a browser.

## How to use

1. Download or clone this repo
2. Open `index.html` in any browser
3. Work through the lessons in order

## What's inside

9 lessons that build on each other:

| # | Lesson | Topics |
|---|--------|--------|
| 1 | Hello, Python! | `print()`, comments |
| 2 | Variables & Types | `int`, `float`, `str`, `bool`, type conversion |
| 3 | Conditionals | `if` / `elif` / `else`, `and` / `or` / `not` |
| 4 | Loops | `for`, `while`, `range()`, `break`, `continue` |
| 5 | Functions | `def`, `return`, default parameters, f-strings |
| 6 | Lists | indexing, slicing, list methods, comprehensions |
| 7 | Dictionaries | key-value pairs, `.get()`, `.items()` |
| 8 | Classes & OOP | `__init__`, `self`, inheritance |
| 9 | Text Adventure Game | putting it all together with `import random` |

Each lesson includes:
- Side-by-side comparisons with JavaScript and Unity C#
- A live code editor to run Python in the browser
- A challenge to complete
- A short quiz

## No install required

Everything runs in the browser using [Pyodide](https://pyodide.org) (Python compiled to WebAssembly). No Python installation needed.
